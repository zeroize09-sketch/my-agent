# ZOXE 網站交付前檢查 (/website-check)

瑞克交付 WordPress / Elementor 網站給客戶前，依序跑這份清單，最後輸出一段可以直接貼給客戶的交付確認訊息。

---

## 使用方式

用戶輸入範例：
- `/website-check`
- `/website-check 睿麗法律事務所`
- `幫我跑網站交付前檢查，客戶是 [名稱]`

如果沒有提供客戶名稱，先問：「請問這次交付的客戶是誰？（用來填入交付訊息）」

---

## 第一步：確認基本資訊

先向瑞克確認以下資訊（沒有的話用 AskUserQuestion）：

1. **客戶名稱 / 公司名稱**
2. **網站網址**（例如 `yourhome.tw`）
3. **主要聯絡表單信箱**（交付訊息要附上）

---

## 第二步：逐項檢查清單

以下每一項，讓瑞克逐一確認（用 AskUserQuestion 跳選項框，或逐行條列讓他快速回答「✅ / ❌ / 待確認」）。

### 🔴 核心必檢（最優先，缺一不可）

| # | 項目 | 說明 |
|---|------|------|
| 1 | **RWD 各裝置顯示正常** | 手機（iPhone / Android）、平板、桌機三種裝置都測過；Elementor 行動版斷點沒跑版 |
| 2 | **表單測試送出 + 信箱有收到通知** | 聯絡表單實際送一封測試信，確認 Gmail / 指定信箱有收到 |
| 3 | **所有 CTA 按鈕 & 連結可以點** | LINE 連結、電話連結、Facebook / IG 連結、頁面內部連結全部點過一次 |
| 4 | **SSL / HTTPS 正常** | 網址列有鎖頭、全站沒有「混合內容」警告 |

### 🟡 品質必檢（影響專業感）

| # | 項目 | 說明 |
|---|------|------|
| 5 | **外掛 + WordPress + Elementor 更新至最新版** | 後台更新中心無待更新項目 |
| 6 | **網站備份已設定** | UpdraftPlus 或 WPvivid 自動備份已開啟、測試備份完成 |
| 7 | **SEO 基礎設定完成** | 每頁 SEO 標題、Meta 描述都填好；圖片有 alt 屬性；Yoast / Rank Math 設定完成 |
| 8 | **速度 & 效能** | 圖片已壓縮（WebP 優先）；快取外掛已啟用；Google PageSpeed 首頁分數 ≥ 70（手機）|

### 🟢 收尾細節

| # | 項目 | 說明 |
|---|------|------|
| 9 | **全站無佔位文字** | 沒有「Lorem ipsum」、「示範文字」、「填入描述」等未替換內容 |
| 10 | **聯絡資訊正確** | 電話、地址、Email、Google 地圖嵌入全部與客戶確認過 |
| 11 | **Favicon 已設定** | 瀏覽器分頁的小圖示是客戶的品牌 Logo |
| 12 | **404 頁面正常** | 輸入錯誤網址會跳到正確的 404 頁面，不會顯示 WordPress 預設錯誤 |

---

## 第三步：統計結果

檢查完畢後，統計：
- ✅ 全過：幾項
- ❌ 未通過：哪幾項（條列）
- ⏳ 待確認：哪幾項

**如果有 ❌ 項目**：先停下來，告訴瑞克「以下項目需要先處理再交付：[列出未通過項目]」。不要繼續產出交付訊息。

**全部 ✅ 才進第四步。**

---

## 第四步：輸出交付確認訊息

全部通過後，產出一段可以直接複製的繁體中文訊息，風格：**專業但有溫度，像 ZOXE 品牌的口氣**。

---

### 範本（依客戶資訊填入）

```
[客戶稱謂] 您好，

您的網站已完成最終確認，可以正式上線了！🎉

以下是我們這次完成的確認項目：

✅ 各裝置（手機 / 平板 / 電腦）顯示正常
✅ 聯絡表單測試通過，信件通知正常接收
✅ 所有連結與 CTA 按鈕功能正常
✅ SSL 安全憑證啟用，全站 HTTPS
✅ SEO 基礎設定完成
✅ 網站自動備份已啟用

網站網址：[網站網址]

接下來您可以開始使用囉！如果日後有任何調整需求或操作上的問題，歡迎隨時聯繫我們。

感謝您這段時間的合作與信任，期待看到您的品牌在網路上發光！✨

ZOXE 零式設計
Rick Z.
```

---

## 迭代提示

用幾次之後，如果覺得某項檢查要加 / 刪 / 改，直接跟莫蒂說：
「幫我改 `/website-check`，把『[項目]』加進去」或「把第 X 項改成 Y」。
