## 瑞克的核心動力模式（2026-05-23 自我盤點）

> 詳見 `daily/2026-05-23.md`。莫蒂在做重大決策建議時要記得這份脈絡。

- **童年線索 → 現在的線**：畫畫 → 設計 → 視覺品牌（一條沒斷）
- **心流時刻**：設計、創造東西的時候
- **耗能時刻**：處理客戶要求的時候
- **核心張力**：ZOXE 的商業模式 =「處理客戶要求 → 產出設計」，跟他的心流模式有結構性衝突
- **Rick Z. 個人品牌**：不只是 ZOXE 引流工具，可能是他「真正想要」的那條線本身
- **室內設計師客群**：邏輯成立（有預算 + 有 know-how），但屬於「正確答案」而非「真心想要」
- **潛水**：喜歡 ≠ 想當教練。目前定位成「Rick Z. 的故事背景而非服務」是對的

### 決策提醒清單

當瑞克問以下問題時，調用上面脈絡：
- 「ZOXE 要不要做 XX 服務？」→ 會增加「處理客戶要求」比重嗎？
- 「Rick Z. 要不要做 XX 內容？」→ 別把它當引流工具，它可能是主場
- 「要不要接 XX 案子？」→ 客戶有預算 ≠ 這條路適合長期投入
- 「要不要往 XX 方向轉？」→ 能讓「創造 > 服務」的比例提高嗎？
